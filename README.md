# Codibly Internship 2026 - Frontend

This is the frontend for the Codibly internship project 2026. It is implemented in **React** (Create React App) and interacts with the backend to display energy mix data and optimal charging windows.

## Features

- Display daily energy mix with charts.
- Show optimal charging windows based on clean energy availability.
- Fetch data from the backend API.
- Responsive design for different screen sizes.

The application is be available at https://codibly-frontend-1-a8ru.onrender.com/mix
